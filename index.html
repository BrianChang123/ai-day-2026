"이런 것도 자동으로 된다고요?"

네, 됩니다. 지금 바로 쓸 수 있는 예시들이에요.

📩 이메일 오면 → 엑셀에 자동 정리
전: 이메일 열고, 복사하고, 엑셀 열고, 붙여넣고...
후: 이메일 도착 → 자동으로 엑셀에 한 줄 추가
📈 매일 아침 → 자동 보고서
전: 아침마다 데이터 모으고, 정리하고, 보고서 만들고...
후: 매일 9시 자동 실행 → 보고서 → Slack 전송
🔔 안 보낸 사람 → 자동 리마인더
전: 하나하나 확인하고, '보내주세요~' 직접 연락...
후: 미제출자 자동 감지 → 자동 알림 발송
🤖 AI가 이메일 → 자동 분류·요약
전: 이메일 읽고, 판단하고, 분류하고...
후: AI가 읽고 → 자동 분류 → 담당자에게 전달
